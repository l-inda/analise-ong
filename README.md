# Sobre
## Requisitos
- Python 3.6
- PIP

## Instalação
```sh
python -m venv venv
pip install -r requirements.txt
```

## Preparação
### Linux
```sh
source venv/bin/activate
```

### Windows
```sh
source venv/Scripts/activate
```

## Execução
### Linux
```sh
python3 main.py
```

### Windows
```sh
python main.py
```
